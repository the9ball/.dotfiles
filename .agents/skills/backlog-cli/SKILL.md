# Backlog CLI

`$HOME/tools/Backlog.Cli/backlog.exe` は、Backlog のプロジェクト・ドキュメント・Wiki を検索・取得できる読み取り専用 CLI です。

詳細な使い方、接続設定、コマンド一覧、出力形式は、exe と同じフォルダの `README.md` を参照してください。

AI エージェントから使う場合は `--json` を優先してください。

API キーはプロンプト、ログ、最終回答に含めないこと。
