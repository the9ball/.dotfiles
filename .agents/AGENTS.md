## Local project instructions

- When an `AGENTS.md` exists in a directory, read `AGENTS.local.md` in the same directory as additional instructions when that file exists.
- AIツールがスクリプトを書く際は、内容を読んで実行を許可してよいか判断しやすくするため、変数名・関数名・コマンド引数などに短縮名（略語・頭字語）をなるべく使わず、意味が明確な名前を使う。

## ファイル変更前の範囲確認

- ソースコード、設定、テスト、ドキュメント、スクリプト、生成物など、ファイルに変更を加える前に、対応内容が決まった段階で変更対象と変更内容を簡潔に提示する。
- 変更対象が1ファイルだけの場合も、この確認とユーザーの承認を省略しない。
- ユーザーが対象や内容を具体的に指定している場合も、変更範囲を簡潔に復唱し、明確な承認を得るまで編集を開始しない。
- 「その範囲で進めてください」「承認します」などの明確な同意を承認とみなす。曖昧な返答だけでは着手しない。
- 承認前でも、調査・既存コードの読解・テスト実行・変更案の作成は行ってよい。
- 承認済みのプランや計画に変更対象と内容が明記されている場合は、それを範囲確認と承認の代わりとしてよい。
- 承認済みの範囲を超える変更が必要になった場合は、追加対象と理由を提示し、再承認を得る。
